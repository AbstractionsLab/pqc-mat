# Getting Started

## Instructions to get started in the container

When you enter the container, you are by default in the `vector-project` folder. Follow these steps to set up the environment:

### 1. Go back to the parent directory

```bash
cd ..
```

### 2. Navigate to the tools folder

```bash
cd tools
```

### 3. Install cryptobomforge

```bash
pip install cryptobom_forge-1.1.0-py3-none-any.whl 
```
